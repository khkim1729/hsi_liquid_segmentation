## 1. 프로젝트 개요 및 초분광 카메라 특성
초분광 이미징(Hyperspectral Imaging, HSI)은 공간(Spatial) 정보와 함께 각 화소별로 수십~수백 개의 파장 대역 스펙트럼(Spectral) 정보를 동시에 수집하는 첨단 진단 기술입니다. 

**Living Optics**의 초분광 카메라 기술은 기존의 라인 스캔(Push-broom) 방식이 지닌 기계적 복잡성과 느린 캡처 속도 한계를 극복하기 위해 **스냅샷 압축 센싱(Snapshot Compressive Sensing)** 기법을 도입했습니다.
* **이중 센서 설계(Dual-Sensor Architecture):** 카메라는 고해상도의 Grayscale/RGB 컨텍스트 이미지 센서와 스펙트럼 인코딩을 위한 코딩 어퍼처(Coding Aperture) 센서를 장착하고 있습니다.
* **동영상 속도 HSI(Video-rate spectral video):** 초당 최대 30프레임(30 Hz)의 실시간 비디오 초분광 촬영을 지원하여, 정적인 피사체뿐만 아니라 흐르는 액체, 움직이는 객체의 spectral signature를 완벽하게 모니터링할 수 있습니다.
* **스파스 샘플링(Sparse Sampling):** 전체 장면 위에 고품질의 스펙트럼 포인트들을 빽빽하게 흩뿌린 형태로 획득하며, 소프트웨어 디코딩 알고리즘을 거쳐 고해상도 장면과 매칭된 점 형태의 스파스 스펙트럼을 제공합니다.

## 2. Living Optics `.lo` (LOFMT) 파일 포맷 계층 구조 및 데이터 형태
`.lo` 확장자는 Living Optics 카메라의 복원 가공(Processed)이 완료된 **spatial-spectral video recording** 파일로, 전용 이진 규격인 **LOFMT**를 따릅니다.
* **LORAW와 LO의 차이:**
  * **`.loraw`:** 카메라 하드웨어 센서에서 수집된 radiometrically 보정되지 않고 인코딩된 이진 무압축 로 데이터입니다.
  * **`.lo`:** 전용 SDK 또는 `analysis-qt` 디코더를 거쳐 보정, 노이즈 제거, 스펙트럼 복원(Spectral Decoding) 및 픽셀 좌표 매핑이 완벽히 가공된 초분광 프레임 비디오 파일입니다.
* **프레임 단위 독립구조:** `.lo` 비디오 파일은 메타데이터와 프레임이 개별적으로 묶인 고정 길이 프레임 블록의 연속적인 병렬 결합체(Concatenation)로 구성되어 있어, 임의 프레임 접근(Random Access)이 용이하도록 고안되었습니다.

### 🌳 계층 구조 분석 명세서 (LOFMT File Layout Hierarchy)
```text
liquid-segmentation.lo (2.42 GB, 총 388개 프레임 순차 배치 구조)
 ├─ frame 0 (6,699,822 Bytes, 고정 바이트 길이)
 │   ├─ frame header (35,630 Bytes, 정밀 헤더 블록)
 │   │   ├─ magic signature (16 Bytes, 캐릭터 배열) : "\x93LOPROCESSEDFMT\x00"
 │   │   ├─ format metadata (17 Bytes, 메타 기술자) : 버전 플래그(1B), 헤더 크기(4B), 데이터 크기(4B) 등
 │   │   ├─ physical metadata (141 Bytes, 이진 변수) : 샘플 수(4B: 4,384), 밴드 수(4B: 96), 해상도 가로/세로 등
 │   │   ├─ 96 spectral band wavelengths list (384 Bytes) : float32[96] (각 물리적 파장 실측값 대역 지정)
 │   │   └─ sparse samples 2D coordinates list (35,072 Bytes) : float32[4384][2] (장면 해상도 내 서브픽셀 2D 좌표쌍)
 │   └─ frame data (6,664,192 Bytes, 순수 데이터 블록)
 │       ├─ high-resolution context image (4,980,736 Bytes) : uint8[2432][2048] (흑백 배경 장면 밝기 행렬)
 │       └─ calibrated reflectance matrix (1,683,456 Bytes) : float32[4384][96] (4,384개 샘플 각각의 96밴드 반사율)
 ├─ frame 1 (6,699,822 Bytes)
 │   └─ ...
 └─ frame 387 (6,699,822 Bytes, 최종 비디오 프레임 블록)
```

### 📊 계층별 실제 데이터 샘플 규격 (Data Format Examples)
계층 구조 내부의 핵심 데이터 변수들이 어떻게 파이썬 넘파이 배열 등으로 리턴되고 채워지는지 실제 사양 예시를 제공합니다.

1. **96개 스펙트럼 밴드 파장 리스트 (`wavelengths`, `float32[96]` 대수적 배열):**
   * **데이터 구조:** 441.00 nm부터 898.25 nm까지의 물리 센서 실측 파장 값이 오름차순 정렬된 1D 실수형 넘파이 배열입니다.
   * **실제 데이터 형태:**
     ```python
     array([441.00, 443.50, 446.00, 448.50, 451.25, ..., 893.25, 895.75, 898.25], dtype=float32)
     ```
2. **스파스 샘플 2D 좌표 목록 (`coordinates`, `float32[4384][2]` 공간적 배열):**
   * **데이터 구조:** 4,384개의 스펙트럼 샘플 포인트가 2D 장면 컨텍스트 상의 어느 지점 `(X, Y)` 픽셀 좌표에 위치하는지를 서브픽셀 정밀도로 기재한 2D 실수형 배열입니다.
   * **실제 데이터 형태:**
     ```python
     array([[ 1559.1234,   497.3456],
            [ 1195.4512,   362.8819],
            [  964.8871,   287.0123],
            ...,
            [   82.3456,  1112.5612]], dtype=float32)
     ```
3. **고해상도 Scene Grayscale 컨텍스트 이미지 (`scene_image`, `uint8[2432][2048]` 이미지 행렬):**
   * **데이터 구조:** 장면의 기하학적 형태 및 텍스처를 묘사하는 가로 2048, 세로 2432 픽셀의 무부호 8비트 정수형 2D 이미지 밝기 행렬입니다.
   * **실제 데이터 형태:**
     ```python
     array([[12, 14, 15, ..., 10, 11],
            [11, 13, 14, ...,  9, 10],
            ...,
            [20, 22, 23, ..., 18, 19]], dtype=uint8)  # 범위: 0 ~ 255 (어두운 실험판 배경에 기인해 전반적으로 낮음)
     ```
4. **초분광 반사율 데이터 행렬 (`spectral_data`, `float32[4384][96]` 분광 반사 강도 행렬):**
   * **데이터 구조:** 4,384개 스파스 샘플 각각이 지닌 96개 파장 대역별 반사율(Reflectance) 강도를 담은 다차원 데이터 행렬입니다.
   * **실제 데이터 형태:**
     ```python
     array([[0.012345, 0.015432, ..., 0.009876],  # 0번 샘플의 96개 밴드 반사 스펙트럼
            [0.034211, 0.031234, ..., 0.021112],  # 1번 샘플의 96개 밴드 반사 스펙트럼
            ...,
            [0.005432, 0.005112, ..., 0.004876]], dtype=float32) # 범위: 0.0 ~ 1.0 (calibrated reflectance)
     ```

## 3. 카메라 하드웨어 셋업 및 압축 센싱 스파스 원리

### ❓ 2432 x 2048 고해상도 장면 위에 왜 4,384개의 스펙트럼 샘플만 존재하는가?
초분광 비디오 카메라는 하드웨어 차원에서 **공간 해상도(Spatial)-분광 채널 수(Spectral)-시간 프레임 레이트(Temporal)** 간의 물리학적 대역폭 및 광량 한계(**Data Cube Bandwidth Constraint**)에 직면합니다. 

만약 모든 픽셀마다 96개의 초분광 대역을 조밀하게(Dense) 수집하고 30 FPS 속도로 전송하려면 초당 대역폭은 다음과 같습니다.
$$\text{Data Rate} = 2048 \times 2432 \times 96 \times 30 \text{ frames} \times 4 \text{ bytes (float32)} \approx 57.34 \text{ GB/s}$$
이는 일반 초분광 카메라 하드웨어가 실시간으로 감당할 수 없는 데이터 인터페이스 및 실시간 처리 연산량의 한계를 한참 초과합니다.

이 문제를 해결하기 위해 Living Optics 카메라는 **스냅샷 압축 센싱(Snapshot Compressive Sensing)** 아키텍처를 적용했습니다. 센서 앞에 마이크로 패턴 코딩 어퍼처(Coded Aperture Mask)를 장착하여 빛을 공간-분광 차원에서 광학적으로 압축 인코딩합니다. 그 후 수학적 복원 디코딩 과정을 거쳐 **"고해상도 Grayscale 컨텍스트 이미지 (2432 x 2048)"**와 **"장면에 격자 배열로 흩뿌려진 4,384개의 고정밀 스펙트럼 스파스 샘플"**을 동시에 추출해 냅니다. 

이를 통해 프레임 속도는 30 Hz의 비디오 주사율을 영구 유지하면서, 전체 이미지 형태와 주요 국소부위의 화학 정보를 빈틈없이 파악할 수 있는 타협점을 실현한 것입니다.

* **카메라 위치 및 각도:** 실험용 플랫 벤치면으로부터 수직 높이 약 50-60cm 상단에 **직하향(Top-Down Nadir View)**으로 단단히 수직 고정되었습니다.
* **대상 물체 및 용기 형태:** 바닥면에 깔린 어두운 무반사 흡광 고무판 위에 **투명하고 납작한 저지 프로필(Low-profile) 유리 용기(Tray)**가 가로로 길게 놓여 있습니다.
* **유체 흐름 거동:** 좌측 영역에서 수성 용액(Aqueous)이 흘러 들어오고 우측 영역에서 유성/솔벤트 액체(Organic Oil)가 스며들며 중앙부에서 대칭적으로 혼합/계면 분할되는 동적 변화 양상을 띱니다.
* **광원 장치:** 직접적인 스펙큘러 핫스팟(조명 반사 눈부심)을 감쇄하고 스파스 반사도 보정 품질을 극대화하기 위해, 용기 좌우 양방향에서 45도 사각(Oblique angle)으로 확산형 라인 광원을 투사하였습니다.

## 4. 초분광 데이터셋 상세 사양 및 물리 파라미터

| 물리/공간/물질 파라미터 | 상세 분석 수치 | 비고 |
| :--- | :---: | :--- |
| **총 프레임 수 (Total Frames)** | **388 프레임** | 총 12.9초 촬영 동영상 데이터 |
| **프레임 레이트 (FPS)** | **30.00 Hz (FPS)** | Video-rate 초분광 레코딩 |
| **물리적 노출 시간 (Exposure)** | **20.00 ms** | 센서 노출 시간 (20,000,000 ns) |

### 🤖 4. 머신러닝 기반 무감독 초분광 세그멘테이션 (Unsupervised Spectral Segmentation)
우리는 픽셀에 임베딩된 96차원의 조밀한 파장 스펙트럼 벡터들을 물리화학적 특징 공간상에서 자동으로 식별하기 위해 **K-Means 무감독 클러스터링 알고리즘($K=3$)**을 프레임 전체 샘플에 적용하여 정밀 세그멘테이션 및 클래스 분류를 수행했습니다.

이 분석을 통해 사전에 정답 라벨링이 존재하지 않는 데이터셋 내에서 **3개의 과학적 유체/배경 클래스**를 성공적으로 감지해내었습니다.

#### 🧠 무감독 분류 모델 예측 및 군집화 작동 원리 (Clustering Mechanism)
K-Means 분류 모델은 다음과 같은 수학적 및 물리화학적 단계로 픽셀을 자동 세그멘테이션합니다.
1. **스펙트럼 차원 벡터 입력:** 각 스파스 포인트(4,384개)에 대하여 441.00 nm ~ 898.25 nm 대역에서 감지된 **96차원의 분광 반사율 벡터(Reflectance Vector)** $\mathbf{x}_i \in \mathbb{R}^{96}$를 입력 특징(Feature) 공간으로 취합니다.
2. **유클리드 거리 최소화 군집화:** 모델은 특징 공간 상에서 중심점(Centroid) $\mathbf{\mu}_k$를 반복 업데이트하며, 각 분광 곡선과 중심점 간의 유클리드 제곱 거리 $d(\mathbf{x}_i, \mathbf{\mu}_k) = \|\mathbf{x}_i - \mathbf{\mu}_k\|^2$를 최소화하는 방향으로 군집화가 수행됩니다.
3. **물리화학적 라벨 할당 논리 (Automated Class Labeling):**
   * **배경 클래스(Background):** 모든 밴드에서 반사 에너지가 `0.005` 내외로 매우 낮고 스펙트럼이 완전히 평탄(Flat low albedo)하여, 반사도 합산 벡터 크기가 최소인 군집을 자동으로 분류합니다.
   * **수용성 액체 클래스(Aqueous Liquid):** 가시광선 대역(VIS)에서 높은 에너지 피크를 띠면서 동시에 **830-840nm 대역의 O-H 제3 오버톤 수산기 흡수 계곡(Absorption Valley)**으로 인해 강한 반사율 드롭(Dip) 현상이 수반되는 군집을 식별합니다. (830nm 반사도를 788nm 반사도로 나눈 감쇄 비율이 최저인 특징 공간 탐지)
   * **유성/유기 솔벤트 클래스(Organic Liquid):** 근적외선 영역(NIR)에서 평탄하고 높은 반사 강도를 계속 유지하며 830nm 영역에서 수산기 흡수 계곡이 전혀 발견되지 않는 소수성 유성 군집을 찾아냅니다.
4. **시계열 예측 일관성 보존 (Temporal Consistency Protection):**
   비디오 프레임이 변할 때 클래스의 색상이 무작위로 바뀌는 현상을 원천 방지하기 위해, **0번 프레임에서 도출된 최적의 3대 물리 중심점(Centroids)을 기준 공간으로 영구 홀딩**합니다. 후속 프레임(1~387)이 로딩되면 base 모델의 중심점을 기준으로 각 분광 샘플 포인트의 소속 군집을 즉각 투영 예측(`kmeans.predict()`)하여, 동적 유체 흐름 속에서도 고유 클래스 및 색상(배경=회색, 수용성=파란색, 유기물=주황색)의 고유 아이덴티티가 100% 일치하도록 보장합니다.

#### 📝 각 K값에 따른 분광학적 해석
* **K = 2 (유체 vs 배경):**
  * **식별 특징:** 가장 단순한 이진 분할로, 알베도(Albedo)가 높은 액체 전체(수성+유기물)를 파란색으로 묶고, 어두운 고무판 및 컵 배경그림자를 회색으로 묶습니다.
  * **한계점:** 물과 유기용매 간의 화학적 차이(830nm 수산기 배음 유무)를 감지하지 못하고, 오직 빛의 총 반사율 세기만을 기준으로 양분합니다.
* **K = 3 (물리/화학적 최적 군집 - Baseline):**
  * **식별 특징:** 배경(회색), 물(파란색), 유성 오일(주황색)의 고유 분자 진동 차이를 정확하게 식별하는 가장 균형 잡힌 모델입니다.
* **K = 4 (배경 음영 및 이중 액체 분할):**
  * **식별 특징:** 물리적 액체 분류(물/오일)는 유지하면서, 무반사 고무판 영역을 **"어두운 깊은 그림자(Dark Shadow, 진회색)"**와 **"약간 빛이 도는 밝은 표면(Bright Surface, 연회색)"**의 두 가지 물리적 영역으로 정밀 세분화합니다.
* **K = 5 (계면 혼합 및 굴절 변화 추적):**
  * **식별 특징:** 가장 극적인 다차원 세그멘테이션입니다! 깊은 음영(짙은 흑색), 밝은 표면(밝은 회색), 물 수용액 코어(파란색), 유성 솔벤트 코어(주황색)와 함께 **"두 유체의 계면 혼합 영역 및 굴절 경계(Liquid Interface / Mixing, 청록색)"**가 독자적인 클래스로 정교하게 도출됩니다! 유체가 합류할 때 일어나는 굴절율 변화 및 과도기적 스펙트럼 혼합 특징을 시각적으로 모니터링하기에 가장 훌륭한 해상도를 보입니다.

## 7. 세밀한 물리/화학적 밴드 분석 (Granular Physical & Chemical Band Analysis)
96개 초분광 밴드 대역(441.00 nm ~ 898.25 nm)을 과학적 파장 물리 및 분자 화학 결합 거동에 맞춰 정밀 분석한 결과입니다. 
본 보고서의 데이터는 액체 세그멘테이션(`liquid-segmentation.lo`) 타겟 데이터셋인 만큼, 수용액 상태, 유기 액체, 그리고 계면 화학 반응 거동을 설명할 수 있는 분자 분광학 분석을 중점적으로 기술합니다.

```
                  441.00 nm                                                       898.25 nm
                      |<------------------- VIS (가시광선) ------------------>|<-- NIR -->|
Wavelength Bands:     [=== Blue ===]  [=== Green ===]  [=== Red ===]  [= RedEdge =]  [=== NIR ===]
Chemical Markers:       Chlorophyll     Transmittance    Chlorophyll     Transition    O-H Overtone
                        & Carotenoids   Optics Peak      Absorption      Reflectance   Water Valleys
```

### 💙 1. 청색 광역 & 유기물 흡수대 (441.00 nm ~ 490.00 nm)
* **스펙트럼 밴드 범위:** Band 0 (441.00 nm) ~ Band 19 (489.50 nm)
* **지배적인 흡수/반사 분자 기작:**
  * **엽록소 강 흡수 대역(Chlorophyll Absorption peak):** 식물성 생화학 물질의 경우 엽록소-b(Chl-b)의 강한 흡수 피크(~453 nm) 및 엽록소-a의 주흡수대(~440-445 nm)가 지배하는 대역입니다.
  * **카로티노이드 및 카로틴(Carotenoids & Carotenes):** 보조 색소 계열(Lutein, $\beta$-Carotene, ~450-480 nm)의 파이 전이($\pi \rightarrow \pi^*$) 전자 전이에 기인한 강한 에너지가 흡수되는 영역입니다.
* **액체 물질 분석 시사점:** 
  * 물속에 용해되어 있는 **용존유기물(Colored Dissolved Organic Matter, CDOM)** 및 액체 내에 포함된 유기 색소 성분의 유무, 혼탁도를 극도로 민감하게 감별해 냅니다. 짧은 파장으로 인해 미세 유탁액의 **레일리 산란(Rayleigh Scattering)** 효율이 극대화되므로 계면활성제나 유탁액의 미세 혼탁도 변화를 관찰하기에 가장 유리한 밴드입니다.

### 💚 2. 녹색 광역 & 광학적 투과 피크 (490.00 nm ~ 570.00 nm)
* **스펙트럼 밴드 범위:** Band 20 (492.25 nm) ~ Band 45 (567.00 nm)
* **지배적인 흡수/반사 분자 기작:**
  * **반사 최대화 구역(Green Reflection Peak):** 광합성 색소의 에너지가 흡수되지 않고 가장 활발히 반사 또는 투과되는 고유 광역입니다. 식생의 경우 녹색 반사 피크(~550 nm)가 형성됩니다.
  * **금속 이온 수화 착물(Metallic Aqua Complexes):** 전이 금속 이온, 특히 $Cu^{2+}$(구리) 또는 $Fe^{2+}$(철) 수화 이온이 결합된 무기 수용액의 경우, $d-d$ 전자 전이 에너지 차이에 의해 약 500~560 nm 대역에서 반사 또는 투영 피크를 관찰할 수 있습니다.
* **액체 물질 분석 시사점:**
  * 액체의 고유한 발색성(Chromaticity) 식별 및 투명 액체가 용기(유리병, 투명 플라스틱)를 통과하여 산란될 때 일어나는 굴절각 별 파장 거동을 추적합니다. 화학 계열별 투명 수용액과 유성 솔벤트 계열 액체의 거시적 분할(Segmentation)의 베이스라인 특징으로 작동합니다.

### 💛 3. 황색-적색 경계 & 색소 지문 대역 (570.00 nm ~ 700.00 nm)
* **스펙트럼 밴드 범위:** Band 46 (570.00 nm) ~ Band 68 (666.00 nm)
* **지배적인 흡수/반사 분자 기작:**
  * **조류 색소 밴드(Algal Pigment Absorption):** 피코시아닌(Phycocyanin, ~620 nm) 및 피코에리트린(Phycoerythrin, ~560-580 nm)과 같은 다세포/단세포 수중 조류의 특이 화학 물질 결합체가 에너지를 강하게 흡수합니다.
  * **엽록소-a 주흡수 극대(Primary Chl-a Peak):** 식물 생체 성분의 최강 광흡수 피크(~675 nm)가 이 대역의 최말단에 칼날처럼 날카롭게 포지셔닝합니다.
* **액체 물질 분석 시사점:**
  * 수성 액체의 환경 오염물 및 미세 조류 번식 유무를 정량화할 수 있습니다. 액체 세그멘테이션 환경에서 음료(콜라, 우유, 오렌지주스 등) 고유의 적색광 투과 스펙트럼과 무기 용매의 적색 투과율 강도 차이를 바탕으로 고정밀 경계 픽셀 구분을 주도합니다.

### 🧡 4. 적색 경계 천이 영역 (700.00 nm ~ 750.00 nm)
* **스펙트럼 밴드 범위:** Band 69 (673.50 nm) ~ Band 76 (733.50 nm)
* **지배적인 흡수/반사 분자 기작:**
  * **레드 엣지 전이(Red-Edge Transition):** 분광분석의 정점이라 평가받는 스펙트럼 급변 대역입니다. 엽록소의 강력한 전자기적 광 흡수가 소실되는 시점과 식물 세포 벽(Spongy Mesophyll)에 의한 근적외선 무한 산란(Scattering)이 개시되는 임계점이 마주하는 교차로입니다.
* **액체 물질 분석 시사점:**
  * 유기 용액의 농도 급변성 및 활성 액체 상태 모니터링에 특화되어 있습니다. 화학적 독성물질 투여 시 식생의 레드 엣지 반사율 변화율($dR/d\lambda$)의 변곡점이 단파장 방향으로 이동하는 **Blue Shift** 현상이 발생하므로, 액체 접촉에 따른 생태 독성학적 차별 진단을 가장 민감하게 해낼 수 있습니다.

### ❤️ 5. 근적외선 & 수분 분자 오버톤 대역 (750.00 nm ~ 898.25 nm)
* **스펙트럼 밴드 범위:** Band 77 (741.00 nm) ~ Band 95 (898.25 nm)
* **지배적인 흡수/반사 분자 기작:**
  * **물분자 O-H 결합 제3 배음 흡수 피크 (3rd O-H Stretching Overtone):** 
    수용성 액체를 정량 분석할 때 가장 강력하고 지배적인 분자 화학 핑거프린트 대역입니다! 물분자는 **~760 nm**와 **830–840 nm**에서 수산기(O-H)의 신축 진동 오버톤(Overtone)에 의해 전자기파 에너지를 흡수하는 특이적 **흡수 계곡(Absorption Valley)**을 형성합니다.
  * **유기 탄화수소 C-H 배음 반응:** 오일이나 알코올 계열 액체 내의 지방족 탄화수소(C-H) 결합의 3차 배음 에너지 흡수가 880–900 nm 인근에서 물과 미세하게 다른 패턴의 굴곡을 형성합니다.
* **액체 물질 분석 시사점:**
  * **물(Water)과 오일(Oil)/솔벤트를 완벽하게 판별해내는 최고의 화학 스펙트럼 필터 역할을 수행합니다!** 수용성 유체는 830-840nm 대역에서 강한 흡수로 반사도가 크게 함몰되는 반면, 소수성 오일류, 가솔린, 글리세린 등은 오버톤 밴드가 전반적으로 완만하여 물과 확실하게 반사율 벡터 수준에서 분기됩니다. 이 근적외선 핑거프린트 덕분에 고성능 머신러닝(Random Forest, SVM) 분류기가 액체 영역을 픽셀 단위로 신속하고 오류 없이 검출해낼 수 있습니다.

* **경로:** `/data/hsi_fm_bench_123/more_projects/all_spectrum_detection_proj/liquid_segmentation/lo_analysis_tool/lo_parser.py`

### 💻 1. 핵심 사용 소스코드 예제

```python
import sys
sys.path.append("./lo_analysis_tool")
from lo_parser import LivingOpticsReader

# 1. 초분광 리더 초기화 (2.42 GB 대용량 파일도 즉시 매핑 완료)
filepath = "./data/spectral-detection/liquid-segmentation.lo"
reader = LivingOpticsReader(filepath)

# 3. 0번 프레임의 전체 고해상도 이미지 및 초분광 스파스 큐브 로딩
frame = reader.get_frame(0)
scene_img = frame['scene_image']      # Shape: (2432, 2048) uint8 이미지 행렬
```

### ⚡ 2. 딥러닝/머신러닝 전처리 파이프라인 연계법

3. **실시간 트래킹 가능:** 초당 30프레임의 풍부한 동영상 스트림으로 수집되어 있으므로, 액체의 단순 분류를 넘어 흐르는 액체의 속도, 확산 계수, 혼합 공정 진단 등 **동적인 유체 시계열 거동 추적(Fluid Dynamics Tracking)** 분석으로 확장이 가능합니다.
